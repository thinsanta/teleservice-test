To start the project i recommend you do the "install -g ." command in the terminal from root directory.
Now you can access the command "hello".

From the root direcotry you can now access some information of the csv file through the command "hello".
Below are the 3 commands you can do to check the info of the list.

commands:
    hello --answer=popular
    hello --answer=car
    hello --answer=companies

popular: Will show you information of the most sold car in each company.
car: Will show you the most sold cars in descending format.
companies: Will show you how many cars each company sold in descending format.
